<heading size="1" align="center">Connecting WiFi</heading>
<hr>
<align type="center"><text type="bold">First make sure you are on this page by choosing non speech option.<br>NOTE: If you use Ethernet Skip this Step.</text></align>
<image>arch-linux-2.png</image>
<hr>
<heading size="4" align="left">Process Begins From Here</heading>
Type This in Your Arch Linux ISO commandline,You will then see something like this.
<code>
iwctl
device list
<image>iwctl-device-list-2-removebg-preview.png</image>
</code>
<collapse heading="If you see off under Powered section " color="#29b8ff" bg-color="#57cdff">
<align type="center"><text color="#29b8ff" type="bold italic">
Type This in your CommandLine.
</text></align>
<code>
iwctl device {device name(in my case wlan0)} set-property Powered on
</code>
<align type="center"><text color="#29b8ff" type="bold italic">
Then Type this in your Command Line to check if it is Now Powerd On.
</text></align>
<code>
<align type="center"><text color="#dbff29" type="bold">device list
</text></align>
</code>
</collapse><hr>

<heading size="10" align="left">If you got "Powered - On" continue from now</heading>
Type this to Scan Your Wifi Networks
<code>
station (device name) scan
station (device name) get-networks
</code>
Now almost there Type this to connect To your Wifi and type passward then.<code>
station (device name) connect (your wifi name)</code>
At last exit from iwd and check your connection
<code>exit
ping -c 5 google.com</code>
If you didn't see any error you are good to go, but if you see do process again.
<hr>
<heading size="8" align="center">Now You Have Successfully Connected to Wifi and successfully connected to WiFi. Now you can continue to next process.</heading>

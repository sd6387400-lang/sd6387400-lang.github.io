In my case I see vda below sr0, in Your case you might see sda, nvme etc. Remember it. And type this in command line
<code>cfdisk /dev/(your disk type in my case vda)</code>So it will be like this.<code>cfdisk /dev/vda</code>

You will then see something like this then.
<image>cfdisk-check-unallocated-disk-space~2.jpg</image>

<box color="#26ff52" type="solid"><heading size="10" align="left">For UEFI Systems</heading>
Now do the following steps
<list type="bullets">
Now select <text color="green">free space</text> From Arrow keys.
Then select New and click Enter
Delete every text and number and Type 1G and enter
Then select that 1Gb Partician just created.
Then goto Type And select EFI File system
Then again select <text color="green">free space</text> again and select New and click Enter 2 times.
Select Type and select linux filesystem
Select Write, type yes and press enter.
</list>
</box>

<box color="#ff4040" type="solid">
<heading size="10" align="left">For MBR Or Legacy Systems</heading>
Follow the following steps:-
<list type="bullets">
Select <text color="green">free space</text> with Arrow keys
Select New and Press Enter 2 times 
Then select the partician you maded right now and select type and select linux filesystem
Select write press enter, type yes and press enter
</list>
</box>

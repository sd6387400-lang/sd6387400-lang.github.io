<box color="#26ff52" type="solid"><heading size="10" align="left">For UEFI Systems</heading>
Now do the following steps
<list type="bullets">
Now select <text color="green">free space</text> From Arrow keys.
Then select New and click Enter
Delete every text and number and Type 1G and enter
Then select that 1Gb Partician just created.
Then goto Type And select EFI File system
Then again select <text color="green">free space</text> again and select New and click Enter 2 times.
Select partician you just made.
Select Type and select linux filesystem
Select Write, type yes and press enter.
</list>
Now type <Code>lsblk</Code> in your command line and see which particians you just made and remember there names like sda4 or nvme04 etc.
And type this 
<code>mkfs.fat32 -F32 /dev/sda5(enter your efi partician instead(1Gb one))
mkfs.btrfs -fL "Arch" /dev/sda4(your linux file system partician(only for SSDs))
mkfs.ext4 /dev/sda4(replace with linux filesystem partician(for HDDs))
<code>mount /dev/sda4(replace with linux file system partician) /mnt
mkdir /mnt/boot
mount /dev/sda5(replace with efi partician(1Gb Partician)) /mnt/boot</code>
<text color="red" type="bold">Warning don't use any otherParticians do wisely cause your windows or any other linux can wipe with it.
</text>
</box>
